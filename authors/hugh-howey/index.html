How AuthorsBookClub would build it

For authors with a clear breakout series, we keep it brutally simple:

  • Hero focuses on the main promise + genre
  • Trilogy/series reading order sits immediately below
  • Optional: mailing list + social links
  • Updates handled for new editions and new releases
